# Consumer Financial Complaints Analytics

## Overview
This project is focused on improving customer satisfaction, and it describes how consumers rate certain products.


## Aims & Objectives
- To evaluate the number of complaints over time
- The company's response rate to the complaints
- Company size relative to the number of complaints

## Steps Taken:
- Data Gathering
- Data Cleaning
- Data Modeling
- Analyze
- Visualization
